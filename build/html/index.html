class bank_account.investment_account.InvestmentAccount(account_number: int, client_number: int, balance: float, date_created: date, management_fee: float)¶

A class representing an Investment Account, inheriting from BankAccount.

This account type has a management fee that is applied to service charges for accounts less than 10 years old.

get_service_charges() → float¶

Calculate the service charges for the account.

Returns:
float: The calculated service charge. For accounts older than 10 years,

only the base service charge is applied. For newer accounts, the management fee is added to the base service charge.

class user_interface.client_lookup_window.ClientLookupWindow¶

A window for looking up client information and managing their bank accounts.

This class inherits from LookupWindow and provides functionality to: - Look up clients by their client number - Display and filter client’s bank accounts - View and update account details

on_filter_clicked()¶

Handle the filter button click event.

Applies or resets the filter on the account table based on the current state.

on_lookup_client()¶

Handle the client lookup process when the lookup button is clicked.

Validates the client number, displays client information, and populates the account table with the client’s accounts.
